# systemd-manage工具
---
systemd-manage是基于systemd服务管理的图形化工具，使用qt图形库开发，提供服务管理，用户会话，配置文件修改，日志查询，性能分析，进程管理等功能。简单方便，进行系统和服务的管理。

# 部署安装
---
使用qmake进行编译配置和安装
```bash
qmake
make
make install
```
